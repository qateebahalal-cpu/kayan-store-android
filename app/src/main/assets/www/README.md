## 📁 هيكل المشروع

```
kayan-store/
├── index.html              # SPA Shell + شاشة تحميل
├── manifest.json           # PWA Manifest
├── sw.js                   # Service Worker Offline
├── README.md               # التوثيق
├── css/
│   ├── main.css            # متغيرات، أزرار، بطاقات، أنيميشن
│   ├── layout.css          # هيدر، سايدبار، فوتر، سلة جانبية
│   └── pages.css           # صفحات المتجر وجميع التخطيطات
├── js/
│   ├── store.js            # حالة التطبيق، سلة، مفضلة، راوتر، بحث
│   ├── renderers.js        # قوالب HTML للصفحات الرئيسية
│   ├── ui.js               # هيدر، سلة جانبية، قائمة متنقلة، فوتر
│   └── pages.js            # صفحات مقارنة، عروض، من نحن، تواصل، تتبع، FAQ
└── data/
    └── products.json       # 24 منتج، 10 فئات، 4 سلايدات، كوبونات، شحن
```
